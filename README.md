### Geofencing Demo

__Description:__

Demo for potential of using Mapbox tools for geofencing applications. The London Congestion Zone surcharges for cars driving into the busiest parts of the city are visualized here with colored dots representing cars. A fictional charge of Â£5 per vehicle per day is used to display total revenue.

Tools used:

- TurfJS
- Mapbox GL JS
- Mapbox Directions API


__Geofencing Demo:__ https://www.mapbox.com/bites/00223/

### Running locally

    npm install
    npm start && open http://localhost:9966/

### Building

    npm run build
